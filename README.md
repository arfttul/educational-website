## Run Locally

Clone the project

```bash
  git clone https://github.com/arafat2201/educational-website.git
```

Go to the project directory

```bash
  cd educational-website
```

Install dependencies

```bash
  npm install
  yarn
```

Start the server

```bash
  npm run start
  yarn start
```
